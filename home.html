Welcome to my Website

HTML


Website ini dibuat menggunakan HTML(Hyper Text Markup Language). HTML digunakan untuk membuat struktur dasar dari sebuah halaman web dengan menghubungkan teks / link antar halaman, memberi struktur dan membagikan informasi yang terkait dengan sebuah halaman web. Konten sebuah web berisi teks, video, audio, gambar.

CSS


Website ini menggunakan CSS untuk mengubah tampilan layoutnya. CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

Javascript


Website ini menggunakan javascript untuk animasi bergerak seperti pada footer di page "About" Website ini. Gunakan JavaScript untuk membutuhkan kontrol yang signifikan atas animasi.
